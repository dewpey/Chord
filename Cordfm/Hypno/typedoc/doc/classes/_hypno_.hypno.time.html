Class Time

Represents a time value such as a timestamp or duration. Time objects are non-opaque mutable objects representing times (either timestamps or durations).

A Time object is represented as a rational number, with a numerator (an int64_t value), and a denominator (an int32_t timescale). Conceptually, the timescale specifies the fraction of a second each unit in the numerator occupies. Thus if the timescale is 4, each unit represents a quarter of a second; if the timescale is 10, each unit represents a tenth of a second, and so on.

https://warrenmoore.net/understanding-cmtime https://developer.apple.com/documentation/coremedia/cmtime-u58?language=objc
