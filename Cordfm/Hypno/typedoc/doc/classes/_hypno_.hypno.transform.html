Class Transform

Represents an affine transformation matrix for use in drawing 2D graphics.

An affine transformation matrix is used to rotate, scale, translate, or skew the objects you draw in a graphics context. The Transform type provides functions for creating, concatenating, and applying affine transformations. CGAffineTransform

inverted

  • inverted(): Transform
    • Defined in hypno.ts:966

    Returns an affine transformation matrix constructed by inverting this affine transform.

    Inversion is generally used to provide reverse transformation of points within transformed objects. Given the coordinates (x,y), which have been transformed by a given matrix to new coordinates (x’,y’), transforming the coordinates (x’,y’) by the inverse matrix produces the original coordinates (x,y).
