Class Asset

Represents a local Asset file on disk. Assets can be of various types, including video, audio, images, kernels, etc. You can find out if an asset is of an audio/visual type using Asset.isAudioVisual.

  • new Asset(path: string): Asset
    • Defined in hypno.ts:74

    Creates an Asset object with the specified path. Paths can be absolute or relative to the main script.js file.

    Throws an exception if the file does not exist at the specified path.

    // create an image asset.
const myImage = new Asset ("./myImage.jpg");

clip

  • clip(timeRange: TimeRange, type?: string): Clip | undefined
    • Defined in hypno.ts:172

    Returns a new Clip object for the specified TimeRange and type from this asset. Possible "type" values: "video", "audio"; Throws an exception if the timeRange is invalid and or beyond this asset's total duration.

    Parameters

    • timeRange: TimeRange

      The source time range.

    • Optional type: string

      The type, possible values: "video", "audio"

      // set the current working timescale.
Time.timescale = 30;

// create an asset.
const overlayAsset = new Asset ("./hypno-logo.mp4");

// create a clip.
const overlayClip = overlayAsset.clip (new TimeRange (new Time (0), overlayAsset.duration));

    Returns Clip | undefined
