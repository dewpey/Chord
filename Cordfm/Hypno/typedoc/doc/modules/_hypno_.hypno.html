Const argv

argv: any[]
  • Defined in hypno.ts:26

The argv property returns an array containing the arguments passed when the main script is evaluated. The first element will be the path to the JavaScript file being executed. The second element will be an array of camera (dynamic) assets (see Asset). The remaining element(s) will be populated with any additional arguments provided by the client application TBD.

Example:

// Grab the first camera asset (second element of argv)
const cameraAsset = argv [1] [0];
