Class Composition

An object that combines media data from multiple file-based sources to present or process media data from multiple sources.

At its top-level, a Composition object is a collection of tracks, each presenting media of a specific media type, e.g. audio or video, according to a timeline. Each track is represented by an instance of Track object. Each track is comprised of an array of clips, represented by instances of Clips. Each clip presents a portion of media data stored in a source container Asset, specified by a path, and a time mapping.

sourceTrackForFrameTiming

sourceTrackForFrameTiming: Track
  • Defined in hypno.ts:426

Gets or sets the source Track object that should be used for the frame timing for this Composition.

If sourceTrackForFrameTiming is not undefined, frame timing for the composition is derived from the source asset's track. This may be used to preserve a source asset's variable frame timing. If an empty edit is encountered in the source track, the compositor composes frames as needed up to the frequency specified in Composition.frameDuration property.
